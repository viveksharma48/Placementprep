Problem : Given an array of size n and an integer k, return the count of distinct numbers in all windows of size k.

Example 1:
Input: arr[] = {1, 2, 1, 3, 4, 2, 3};
       k = 4
Output: 3 4 4 3

Example 2:
Input: arr[] = {1, 2, 4, 4};
       k = 2
Output: 2 2 1

Source: This problem has appeared in Microsoft Interview Question.
